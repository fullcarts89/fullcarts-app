# FullCarts YouTube Strategy — Shorts Funnel + Long-Form Authority

**Date:** June 13, 2026
**Parent:** `docs/plans/2026-06-10-face-forward-content-strategy.md` · companion to `posting-schedule.md`
**Why this exists:** YouTube is a different machine from TikTok/Reels and needs its own layer. First
real signal: **10 subscribers from Shorts** — the funnel works; now give those subs a *reason to stay*.

---

## The core insight: YouTube is a search engine, not a feed
| | TikTok / Reels | YouTube |
|---|---|---|
| Discovery | FYP, *now* | **Search + Suggested, for months/years** |
| Content life | days | **evergreen (compounds)** |
| Rewards | hooks, volume | **depth, watch-time, authority** |
| The subscribe | low-commitment | **a subscriber expects a content *home*** |

**The unlock:** a 2,200-entry database = **2,200 long-tail search queries we can own.** Someone Googles
*"did Folgers get smaller"* → that's a YouTube result, forever. No other format we make compounds like
this. YouTube is where the database is the *strongest* asset, not just a source.

---

## The 3-layer model

1. **Shorts = the discovery funnel (no change).** Keep cross-posting the Mon/Wed/Thu/Fri filmed shorts
   (`series.md` formats). This is what's already pulling subs. YouTube Shorts caption rules live in
   `profile-copy.md` (≤3–5 topic hashtags, `#Shorts`, no `#shrinktok` — see "hashtags per platform").
2. **Long-form = the authority home (the new layer).** ≥1/month committed. Searchable, evergreen, the
   reason a subscriber sticks around. This is the YouTube-native expansion.
3. **Community tab = warm the subs between uploads.** Where carousels live on YouTube: **image posts +
   polls** (eligibility threshold has been dropping toward all channels — enable when available).
   Repurpose the DB carousels + story polls here; no new work.

## Long-form: the 5 repeatable formats (≥1/month)
Start with **#1** — lowest effort (screen-record + VO), highest search compounding.

| # | Format | Title pattern (matches search queries) | Length | Source |
|---|---|---|---|---|
| 1 | **Single-brand investigation** | *"Is [Brand] Getting Smaller? (I Checked the Data)"* | 4–8 min | one entity + its events; expand a *Caught:* short |
| 2 | **The Tier List (full)** | *"I Ranked Every [Category] Brand by Shrinkflation"* | 8–15 min | the carousel `TierList`, expanded; rankings = watch-time magnet |
| 3 | **The data deep-dive** | *"I Analyzed 2,200 Shrinkflation Cases. Here's What I Found."* | 8–12 min | whole-DB rollup; the video that *justifies the channel* |
| 4 | **Recap** | *"The Worst Shrinkflation of [Month / Year]"* | 5–10 min | top events by window; recurring + evergreen |
| 5 | **Evergreen explainer** | *"How to Never Get Fooled by Shrinkflation Again"* | 5–8 min | methodology + consumer-defense; pure search magnet |

**Cadence:** 1/month to start (sustainable with a day job). Pick the brand/category off the same
convergence brief that feeds the Wed *Caught:* — so the long-form is the *deep* version of a topic the
Shorts already teased. Scale toward 2×/month only if retention holds.

## The "I don't have time" production hack
Long-form *sounds* like 10× the work; the database makes it ~2×:
- **Screen-record fullcarts.org + voiceover** — data-dense long-form with minimal filming; your face
  just bookends it (open + CTA). The DB does the visual heavy lifting.
- **Compile your Shorts** — 5–6 related Shorts + connective VO = one long-form. Reuse, don't recreate.
- **Expand a *Caught:*** — the Wed short is the seed; the long-form is the full investigation.
- Same evidence rules apply (three-bucket policy) and the **dating convention** (`visual-identity.md`):
  stamp current figures, re-verify on upload day.

## Search & packaging (YouTube-specific)
- **Titles are search queries.** Write what people *type*: "Is X getting smaller", "X shrinkflation",
  "why is X so expensive". Front-load the brand/keyword. (vs. TikTok, where the hook is spoken.)
- **First line of the description = the keyword sentence** + the date line (`profile-copy.md` "Dating").
- **Thumbnails:** the `Thumbnail` comp look (face + `−X%` + `CAUGHT:`), consistent so the channel grid
  reads as one body of work.
- **Hashtags:** topic/search tags only (`#shrinkflation #[brand] #shorts`), ≤3–5. Community tags like
  `#shrinktok` are TikTok/IG-only — they have no search volume here.

## Playlists & session time (free growth lever)
- Build a **"Caught:" playlist** now — one view autoplays into the next; session watch-time is YouTube's
  #1 ranking lever and costs zero new content.
- Group by series/category (Caught: · Tier Lists · Explainers) so a searcher who lands on one bingeing.
- **End screens + pinned comment** point to the next video / fullcarts.org.

## Metrics that matter here (different from TikTok)
- **Average view duration / retention** (not just views) — YouTube ranks on watch-time.
- **Search vs. Suggested vs. Shorts-feed traffic** (in Analytics) — confirms the evergreen flywheel.
- **Subscribers per video** — the "is this a home?" signal. Long-form should out-convert Shorts.
