**Purpose:** a full year of topics for the Sunday persona vlog, so the slot is never a blank page.
**These are premises, not scripts** — each gets a unique ~60s take, but the **format is locked**
(human moment → mission → soft CTA; see `series.md`). Persona + facts must obey `content-rules.md`,
`approved-claims.md`, and the persona anchor ("tired dad, sick of getting ripped off, built the
largest free public shrinkflation database; does data for a company billions have used — never named").

**How to use:** pick one per week, check it off. Rotate themes (≈1 from each theme every 6 weeks) so
no two consecutive Sundays share a vibe. Refill the backlog before it empties.
